## Description
A fast, diffusion-powered large language model platform built for parallel token generation. Provides high‑throughput, low‑cost inference, fine‑grained output control, and multimodal integration.

## Features
- Parallel token generation (diffusion LLM)
- 2‑3× faster than autoregressive models
- < 50 % cost per token
- Schema‑aware output enforcement
- Unified multimodal support (text, audio, image, video)
- Easy integration via Rust crate and CLI
