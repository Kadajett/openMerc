# OpenMerc Project

OpenMerc is a fast, diffusion‑based large language model platform. It provides parallel token generation, lower latency, and fine‑grained output control.

## Features
- Parallel token generation (multiple tokens per step)
- Reduced inference cost (≈50% of traditional auto‑regressive models)
- Fine‑grained output constraints
- Easy integration with existing ML pipelines

## Getting Started
```bash
# Clone the repository
git clone https://github.com/yourorg/openmerc.git
cd openmerc

# Install dependencies
pip install -r requirements.txt

# Run the demo
python -m openmerc.demo
```
